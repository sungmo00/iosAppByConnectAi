## 📐 4. 그리드 & 스페싱 시스템

### 4-1. 8pt 그리드
```
Base Unit: 8px

Spacing Scale:
- XS: 4px   (밀집, 아이콘 내부)
- S:  8px   (컴포넌트 내부)
- M:  16px  (섹션 간격 기본)
- L:  24px  (화면 섹션 간격)
- XL: 32px  (큰 분리)
- XXL: 48px (화면 간격)
```

### 4-2. 카드 패딩
| 카드 유형 | 패딩 (상·하·좌·우) |
|-----------|---------------------|
| Full Card | 16px / 16px / 16px / 16px |
| Compact Card | 12px / 12px / 12px / 12px |
| Tall Card | 20px / 20px / 16px / 16px |

### 4-3. 모서리 반경 (Border Radius)
| 유형 | 반경 | 용도 |
|------|------|------|
| `Small` | 8px | 버튼, 작은 카드 |
| `Medium` | 12px | 일반 카드 |
| `Large` | 16px | 큰 카드, 바텀 시트 |
| `Full` | 50% | 아바타, 아이콘 컨테이너 |

## 🧭 5. 하단 탭 내비게이션 (Tab Bar)

### 5-1. 기본 구조
```
┌─────────────────────────────────┐
│                                 │
│         Main Content            │
│                                 │
├─────────────────────────────────┤
│  🏠 Home    🔍 Search   ⚙️ Settings  │ ← Tab Bar
└─────────────────────────────────┘
```

### 5-2. 탭 설정 (3~5개 권장)
| 순서 | 이름 | 아이콘 (SF Symbols) | 화면 경로 |
|------|------|---------------------|-----------|
| 1 | Home | `house.fill` | `/home` |
| 2 | Search | `magnifyingglass` | `/search` |
| 3 | Explore | `compass.fill` | `/explore` |
| 4 | Favorites | `heart` | `/favorites` |
| 5 | Settings | `gear` | `/settings` |

### 5-3. Tab Bar 스타일
| 속성 | 값 |
|------|-----|
| 높이 | 88px (아이폰 홈 인디케이터 공간 포함) |
| 배경 | `#FFFFFF` + 그림자 (`y: -2, blur: 10, opacity: 0.1`) |
| 선택 상태 색상 | `#007AFF` (아이콘 + 텍스트) |
| 비선택 상태 색상 | `#8E8E93` |
| 아이콘 크기 | 24px × 24px |
| 텍스트 | SF Pro Regular 10pt (Small) |

## 🖼️ 6. 메인 화면 컴포넌트 구조

### 6-1. 헤더 (Header)
```tsx
┌─────────────────────────────┐
│  [Logo]   [User Avatar]     │ ← Left: 브랜드, Right: 프로필
│  "Welcome Back"             │ ← H2 제목
└─────────────────────────────┘

스타일:
- 높이: 60~80px
- 배경: Gradient (`#007AFF` → `#5856D6`) 또는 흰색
- 패딩: 16px / 20px / 16px / 16px
```

### 6-2. 카드 (Card)
```tsx
┌─────────────────────────────┐
│  [Icon/Thumbnail]           │
│                             │
│  Title (H3)                 │ ← SF Pro Semibold 22px
│  Description (Body)        │ ← SF Pro Regular 15px
│                             │
│  [CTA Button] ───────────→  │
└─────────────────────────────┘

스타일:
- 배경: #FFFFFF
- 그림자: 0 2px 8px rgba(0,0,0,0.08)
- 테두리 반경: 12px
```

### 6-3. 버튼 (Button)
| 유형 | 스타일 | 용도 |
|------|--------|------|
| Primary | 배경 `#007AFF`, 텍스트 흰색 | 주요 액션 (로그인, 구매) |
| Secondary | 배경 투명 + 테두리 `#007AFF` | 보조 액션 |
| Tertiary | 텍스트만 `#007AFF` | 링크형 버튼 |
